## Prerequisite
- python --version 3.10.12
```shell
cd ./Intel-Edge-AI-Project/ITS_YOUR_TURN_CLIENT
python -m venv iyt_env
source iyt_env/bin/activate
(iyt_env) pip install -U pip
(iyt_env) pip install -r requirements.txt
```

